# PostgreSQL 正體中文使用手冊

本專案用於編譯 PostgreSQL 官方使用手冊之正體中文版  
主要服務台灣的 PostgreSQL 使用者  
詳情請到: [https://pgsql-tw.gitbooks.io/documents](https://pgsql-tw.gitbooks.io/documents)

目前編譯的語言及版本為:

* 台灣正體中文, PostgreSQL 10
