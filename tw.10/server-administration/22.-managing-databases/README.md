正在執行的 PostgreSQL 伺服器的每個執行程序都會管理一個或多個資料庫。而資料庫是組織 SQL 物件（資料庫物件）的最高層級。本章介紹資料庫的屬性以及如何建立、管理和銷毀資料庫。
